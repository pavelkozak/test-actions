# VPe04 — тестовый бэкенд на FastAPI

Простое API: проверка работоспособности и текущее время сервера в UTC.

## Требования

- Python 3.10+

## Установка

```bash
python -m pip install -r requirements.txt
```

## Переменные окружения

Файл `.env` подхватывается через `python-dotenv` при старте приложения. Пример значений — в `env.example`:

| Переменная | По умолчанию | Описание |
|------------|----------------|----------|
| `HOST`     | `0.0.0.0`      | Адрес привязки uvicorn |
| `PORT`     | `8000`         | Порт |

## Запуск

```bash
python main.py
```

Либо напрямую:

```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

После старта откройте интерактивную документацию: [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs).

## Эндпоинты

| Метод | Путь      | Описание |
|-------|-----------|----------|
| GET   | `/health` | Статус сервиса |
| GET   | `/time`   | Текущее время сервера (UTC): ISO 8601 и Unix-секунды |

### Примеры ответов

`GET /health`

```json
{"status":"ok"}
```

`GET /time`

```json
{
  "utc_iso": "2026-04-03T12:00:00.123456Z",
  "unix": "1775217600"
}
```
